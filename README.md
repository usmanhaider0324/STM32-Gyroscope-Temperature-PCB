# STM32F411 Gyroscope & Temperature Sensor PCB

## 📌 Overview
This repository contains the complete hardware design of a 4-layer STM32-based embedded system integrating motion and temperature sensing. The board is built around the STM32F411CEU6 microcontroller and interfaces with an MPU6050 gyroscope/accelerometer and ADT7420 temperature sensor.

The project demonstrates professional PCB design practices including proper power regulation, controlled stackup planning, signal routing, and fabrication documentation.

---

## 🛠 Key Features
- STM32F411CEU6 (ARM Cortex-M4) microcontroller
- MPU6050 Gyroscope & Accelerometer (I2C)
- ADT7420 High-Accuracy Temperature Sensor
- USB interface with AMS1117-3.3V LDO regulation
- 4-Layer PCB Stackup (1.66mm total thickness)
- Dedicated GND planes for signal integrity
- SWD Programming Interface
